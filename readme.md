#### How to use

npm install
python -m pip install -r requirements.txt

for running

```
npx ts-node wather/wather.ts
python wather/wather.py
php wather/wather.php

python range/range.py
npx ts-node range/range_1_100.ts
php range/range_1_100.php
```
